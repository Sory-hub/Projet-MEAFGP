# Projet-MEAFGP
Ce projet consiste à construire et analyser un portefeuille optimisé composé de 10 actions britanniques du FTSE 100, sur la période 2013–2022, en appliquant les modèles classiques de la théorie moderne du portefeuille. Un suivi de performance sur le mois de mars 2026 vient compléter l'analyse historique.
